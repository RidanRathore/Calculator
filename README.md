# Text-Based Calculator 🧮

A simple Python calculator built from scratch.  
Supports basic operations: addition, subtraction, multiplication, and division.  
Includes error handling for invalid inputs and division by zero.

---

## Features
- Continuous use until quit (`q`, `quit`, `exit`, `n`)
- Multiple operation inputs (`+`, `add`, `-`, `subtract`, etc.)
- Error handling for non-numeric inputs
- Friendly user prompts 😁

---

## Future Upgrades
- Calculation history
- Advanced math functions (power, sqrt, modulus)
- GUI version using Tkinter
